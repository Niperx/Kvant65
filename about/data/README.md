# Контент со старого сайта РРЦ «Кванториум»

Файлы сюда скопированы из сохранённых страниц kvantorium.sakhalin.gov.ru для вставки в Wagtail.

## Как использовать

1. **Основные сведения (О центре)**  
   Создайте страницу типа «О нас (текстовая страница)», заголовок «Основные сведения» или «О центре», slug например `osnovnye-svedeniya` или оставьте в разделе «О нас».  
   Откройте `osnovnye_svedeniya_body.html`, скопируйте содержимое (без первого комментария) и вставьте в поле «Текст страницы». Если в редакторе есть переключение в режим HTML/исходника — вставьте туда.

2. **Мобильный комплекс**  
   Страница типа «О нас (текстовая страница)», заголовок «Мобильный комплекс», slug `mobilnyy-kompleks` (или добавьте как дочернюю к «О нас»).  
   Вставьте содержимое из `mobilnyy_kompleks_body.html` в поле «Текст страницы».

3. **Расписание занятий**  
   Создайте страницу типа «Расписание занятий», заголовок «Расписание занятий», slug `raspisanie`.  
   В поле «Вступление» можно ввести: *Актуальное расписание занятий на 2025–2026 учебный год.*  
   В поле «Таблица расписания (HTML)» вставьте содержимое из `raspisanie_table.html` (без первого комментария). Полную таблицу при необходимости возьмите из сохранённого файла «Расписание занятий — РРЦ Кванториум.html».

4. **Документы**  
   Страницу «Документы» создайте той же моделью «О нас (текстовая страница)» и заполните вручную ссылками на документы или загрузите файлы через «Документы» Wagtail и добавьте ссылки в текст.

## Структура в меню

- О нас → О центре (основные сведения), Сотрудники, Документы, Мобильный комплекс  
- Расписание — отдельный пункт меню (корневая или под «О нас» по желанию)
